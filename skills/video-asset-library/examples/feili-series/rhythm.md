# 飞栗系列 · 节奏曲线(90s @ 30fps = 2700 帧)

> 信息密度标注:🟢 低(画面可复杂) 🟡 中 🔴 高(画面必须稳)

## 起(0–8s / 帧 0–240)🟢
- 钩子:客观事实型焦虑点(效率差几十倍 / 两人团队接十人活)
- 大字标题 + 主色关键词,配音一句话点题

## 承(8–30s / 帧 240–900)🟡
- 场景展开:这个工具/用法能干什么(开眼界)
- 每 5-6s 一个信息点,画面跟配音走,不抢

## 转(30–70s / 帧 900–2100)🔴
- 手把手:每步操作可复刻(装环境 / 输命令 / 点哪里)
- **信息密度最高段**:画面必须稳定,命令/参数用等宽体定格 ≥2s
- 关键命令处给音效强调(whoosh 轻量)

## 合(70–90s / 帧 2100–2700)🟢
- 金句收尾(大标题居中)+ 系列回顾注脚(小字灰)
- 引导动作:关注 / 看上一期 / 评论区领资料

## 节奏红线
- 不允许连续两个 🔴 段(观众处理不过来)
- 每 15s 内必须有一个视觉焦点变化(防划走)
- 转场默认硬切;情绪断裂才用特效转场
