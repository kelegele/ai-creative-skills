# 配图设计系统

复用卡片 agentic 设计系统 + 公众号排版规范,映射到配图场景。

## 预设 1:Agentic / 飞栗(默认)

与卡片、公众号排版同源。

### 配色

| Token | 色值 | 用途 |
|---|---|---|
| Primary | `#FF5701` | 强调、箭头、高亮数字 |
| Text | `#111827` | 正文 |
| Text-2 | `#374151` | 次级正文 |
| Subtitle | `#6B7280` | 说明、图注 |
| Tertiary | `#9CA3AF` | 标签、最弱信息 |
| Surface | `#FFFFFF` | 卡片底、背景 |
| Surface-2 | `#F6F6F1` | 标签底、引用底 |
| Success | `#16A34A` | 正向(✓) |
| Warning | `#D97706` | 提示(⚠️) |
| Danger | `#DC2626` | 风险(✗) |

### 字体

- 标题/数字大字:Noto Serif SC(衬线,与卡片 Playfair 气质统一),weight 700
- 正文/说明:Noto Sans SC + Manrope,weight 400
- 强调:weight 700 + 主色 `#FF5701`

### 配图场景组件

| 组件 | 规范 |
|---|---|
| 卡片底 | Surface `#FFFFFF`,圆角 12px,必要时 1px `#E5E7EB` 描边 |
| 流程图箭头 | 主色 `#FF5701`,2px,三角箭头 |
| 标签/徽章 | Surface-2 `#F6F6F1` 底 + Text 字,圆角 4px |
| 图标占位 | emoji 或线条图标,主色描边 |
| 截图风(收件箱等) | 灰阶为主 + 主色点睛(红点/高亮),简化真实界面 |
| 数据/数字 | Serif 大字 + 主色 |

### 反模式

- 不用渐变 / 阴影 / 毛玻璃(违背冷静陈述者)
- 不堆砌 emoji(每图最多 3-4 个,信息性)
- 不用主色铺大面(只做强调/箭头/边框)
- 截图风不照搬真实 UI 细节,简化到能秒懂

## 其他预设(预留,初版不实现)

候选:Minimal/极简、Warm/暖调、Tech/科技。结构与 agentic 预设同(配色/字体/组件/反模式),新增加块即可。

## 预设切换

第 0 步问用户选预设,默认 agentic。切换 = 换设计 token 集,HTML 生成逻辑不变。
