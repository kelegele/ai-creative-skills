# 占位写法范例

范例展示 article.md 里配图占位的标准写法,配图 skill 按此解析。

## 范例文章片段

```markdown
# AI 有了自己的邮箱

你的邮箱堆着几百封 newsletter。

> 🖼️ 【封面 · 900×383 · 生成】左右两邮箱并排,左人类邮箱堆满未读灰暗,右 AI 邮箱干净明亮,中间「隔离」分隔线。

正文段。

> 🖼️ 【配图① · 1080×自由 · 生成】收件箱界面,500+ 未读 newsletter,红点密布,灰冷压抑。

正文段。

> 🖼️ 【配图② · 1080×自由 · 自供】images/inbox.png

结尾。
```

## 草稿态(回填后)

占位块保留,图片插入其下方:

```
> 🖼️ 【封面 · 900×383 · 生成】...

![封面](images/gzh-imaget-封面.png)

> 🖼️ 【配图① · 1080×自由 · 生成】...

![配图①](images/gzh-imaget-配图①.png)

> 🖼️ 【配图② · 1080×自由 · 自供】images/inbox.png

![images/inbox.png](images/inbox.png)
```
