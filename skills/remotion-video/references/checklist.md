# Remotion 视频生成前置检查清单

> 每次开始制作 Remotion 视频前,逐项确认以下所有原子元素,避免遗漏导致成片质量下降。未确认项标"待补充",不得跳过。

## 一、可见层检查(素材与内容)

### 1. 镜头(Shot)
- [ ] 所有镜头素材是否已按场景/镜号/Take 命名归档?
- [ ] 是否已完成镜头库筛选(可用/备用/废片)?
- [ ] 每条镜头的入点、出点是否已标记?
- [ ] 多机位素材是否已完成同步对位?
- [ ] 代理文件(Proxy)是否已生成并可流畅预览?

### 2. 脚本(Script)
- [ ] 分镜脚本(Storyboard)是否已确认最终版?
- [ ] 每一场戏的叙事功能是否明确?
- [ ] 台词/旁白文本是否已定稿并标注时间码?
- [ ] 是否有备用方案应对脚本未覆盖的拍摄意外?

### 3. 转场(Transition)
- [ ] 场景之间的衔接方式是否已规划(硬切/叠化/匹配剪辑/特效转场)?
- [ ] 转场是否服务于叙事或情绪,而非单纯炫技?
- [ ] 越轴镜头是否有缓冲或故意设计?
- [ ] 转场时长是否已标注(帧数)?

### 4. 动效(Motion Graphics)
- [ ] 标题/字幕/数据可视化等动效需求是否已列出清单?
- [ ] 动效风格是否与整体视觉调性一致?
- [ ] 动效节奏是否与剪辑节奏匹配?
- [ ] 字体、颜色、安全框位置是否已确认?

## 二、隐形层检查(操控观众注意力)

### 5. 时间/节奏(Timing & Pacing)
- [ ] 整场视频的情绪张力曲线是否已规划(起-承-转-合)?
- [ ] 是否存在连续高潮或连续平淡的段落?
- [ ] 每个镜头的时长是否服务于节奏,而非随意设定?
- [ ] 关键信息点是否留有足够的"阅读时间"?

### 6. 空间/方位(Spatial Orientation)
- [ ] 场景地理关系是否清晰(观众不会迷路)?
- [ ] 180° 轴线是否遵守?越轴是否有明确目的?
- [ ] 人物位置、视线方向是否连贯?
- [ ] 镜头运动方向(推/拉/摇/移)是否流畅衔接?

### 7. 声音(Audio Layer)
- [ ] 对白/旁白是否已降噪、去混响、音量平衡?
- [ ] 环境音/拟音是否已补充完整?
- [ ] 音乐的情绪走向是否与画面匹配?
- [ ] J-cut / L-cut 的使用点是否已标注?
- [ ] 静默段落是否故意设计(而非音频缺失)?

### 8. 视线/注意力引导(Eye Tracking)
- [ ] 每个画面的视觉焦点是否明确?
- [ ] 剪辑点是否跟随人物视线或动作顶点?
- [ ] 是否需要通过构图或运动引导观众看向特定区域?
- [ ] 信息密集时画面是否足够稳定?

### 9. 表演节拍(Performance Rhythm)
- [ ] 演员的呼吸、眨眼、微表情是否被完整保留?
- [ ] 反应镜头(Reaction Shot)是否已选取最佳版本?
- [ ] 剪辑点是否切在"气口"而非表情半道?

### 10. 信息密度(Information Density)
- [ ] 单位时间内观众需要处理的信息量是否合理?
- [ ] 视听是否同时过载(画面复杂 + 对话密集)?
- [ ] 关键信息是否有视觉或听觉的强调处理?

## 三、技术层检查(Remotion 工程)

### 11. 工程配置
- [ ] Remotion 版本是否已确认(4.x / 5.x)?
- [ ] 分辨率、帧率、时长是否已设定?
- [ ] 色彩空间是否正确(Rec.709 / DCI-P3 / Rec.2020)?
- [ ] 输出格式是否已确认(MP4 H.264 / ProRes / WebM)?

### 12. 资产准备
- [ ] 所有图片/视频/字体/音频是否已放入 `public/` 目录?
- [ ] 资产路径是否使用 `staticFile()` 正确引用?
- [ ] 字体是否通过 `@remotion/google-fonts` 或本地加载引入?
- [ ] SVG/矢量图形是否已优化(无冗余节点)?

### 13. 动画规范
- [ ] 动画优先使用 `spring()` 和 `interpolate()`,避免线性僵硬?
- [ ] 所有时间描述是否已转换为帧数(避免"2秒后"模糊表达)?
- [ ] 组件是否已原子化拆分(可复用、可调试)?
- [ ] 参数是否已外置(颜色、速度、缓动函数提到 props 或常量文件)?

### 14. 代码质量
- [ ] TypeScript 类型是否正确(无隐式 any)?
- [ ] `useCurrentFrame()` 返回值是否在 `0` 到 `durationInFrames-1` 范围内?
- [ ] 是否已添加帧级调试注释?
- [ ] 是否有明确的"可替换区域"标记(`// ASSET: 替换为实际产品图`)?

## 四、快速决策卡

| 场景 | 默认选择 | 例外情况 |
|------|----------|----------|
| 转场 | 硬切 | 时间跳跃用叠化,空间转换用匹配剪辑,情绪断裂用特效转场 |
| 动画缓动 | `spring({stiffness: 100, damping: 15})` | 机械感需求用线性,庄重感用 easeInOut |
| 音频层级 | 对白 > 音效 > 音乐 | 音乐主导段落(MV/预告片)可提升音乐层级 |
| 信息呈现 | 画面稳定 + 音频强调 | 快节奏段落可画面复杂 + 音频极简 |
| 色彩空间 | Rec.709 | 院线/高端广告用 DCI-P3,HDR 内容用 Rec.2020 |
